# wincab
Windows内置制作CAB文件的小工具iexpress，MakeCAB-GUI版可实现多个文件压缩成一个cab包

此工具windows系统已经自带了，Win+R运行框输入 iexpress 即可打开，或者在桌面双击打开仓库中的iexpress.exe
